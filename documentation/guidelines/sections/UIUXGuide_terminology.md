# Terminologie

| Begriff <br>DE \/ EN                                 | Erläuterung                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| :--------------------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <b>Kopfzeile / Header</b>                            | Kopfbereich der Kartenanwendung, in dem sich Funktionen zur Steuerung der Kartenanwendung, als auch Titel und Logo befinden.                                                                                                                                                                                                                                                                                                                                                                                                                  |
| <b>Kartenauswahl / Map selection</b>                 | Bereich in der Kopfzeile in dem die Kartenansicht ausgewählt werden kann.                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <b>Inhaltsauswahl / Content selection</b>            | Bereich in der Kopfzeile über den die Elemente zur Steuerung / Beschreibung der Inhalte ausgewählt werden können (Inhaltsfenster, Mein Arbeitsbereich, Legende).                                                                                                                                                                                                                                                                                                                                                                              |
| <b>Inhaltsfenster / Content window</b>               | Darstellung einzelner Inhaltselemente, wie Karten- oder Planungsebenen und Ansichtspunkte.                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| <b>Mein Arbeitsbereich / My Workspace</b>            | Auflistung von Ergebnissen aus Werkzeugen oder Analysen. Ein Ergebnis kann durch ein Werkzeug automatisch oder manuell durch den Nutzer in "Mein Arbeitsbereich" übernommen werden.                                                                                                                                                                                                                                                                                                                                                           |
| <b>Legende / Legend</b>                              | Darstellung der Kartensignaturen (Symbole, Farbschema, Linienausprägung) für angezeigte Kartenebenen innerhalb der Kartenansicht.                                                                                                                                                                                                                                                                                                                                                                                                             |
| <b>Werkzeugkästen / Toolboxes</b>                    | Befinden sich in der Kopfzeile der Kartenanwendung und stellen thematische Werkzeugleisten zur Verfügung. Beispiele dafür sind der Standard-Werkzeugkasten ([Toolbox](https://gitlab.virtualcitysystems.de/vcsuite/npm/vcmap/ui/-/blob/feature-vue3vuetify3/documentation/TOOLBOX.md?ref_type=heads#SelectToolboxComponent)) oder der Planner, welcher einen eigenen Werkzeugkasten für Planungen bereitstellt.                                                                                                                               |
| <b>Werkzeugleiste / Toolbar</b>                      | Befindet sich im oberen Bereich der Kartenansicht und erlaubt den Zugriff auf alle Werkzeuge eines thematischen Werkzeugkastens.                                                                                                                                                                                                                                                                                                                                                                                                              |
| <b>Werkzeug / Tool</b>                               | Ist ein (einzelnes) Werkzeug, welches sich in der Werkzeugleiste der Karte befindet und spezifische Funktionalitäten zur Interaktion mit Kartenansicht/-inhalten bereitstellt. Ein Werkzeug kann ein einzelnes Werkzeug oder eine thematische Gruppe von Werkzeugoptionen (z.B.: Zeichenwerkzeuge oder Auswahlwerkzeuge) bilden. Siehe [SingleToolboxComponent, GroupToolboxComponent](https://gitlab.virtualcitysystems.de/vcsuite/npm/vcmap/ui/-/blob/feature-vue3vuetify3/documentation/TOOLBOX.md?ref_type=heads#SelectToolboxComponent). |
| <b>Allgemeine Funktionen / General Functions</b>     | Bereich der Kopfzeile (rechts) in dem grundsätzliche Kartenfunktionen (Teilen / Drucken, Suche, Hauptmenu) zu finden sind.                                                                                                                                                                                                                                                                                                                                                                                                                    |
| <b>Kartenansicht / Map view</b>                      | Hauptbereich der Kartenanwendung, in dem die Geodaten dargestellt werden.                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <b>Navigationsbereich / Navigation section</b>       | Befindet sich in der Kartenansicht und gruppiert alle Elemente die der Navigation in der Kartenansicht dienen.                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <b>Fußzeile / Footer</b>                             | Fußzeile der Kartenanwendung in der die Positionsanzeige, Impressum, Datenschutzhinweise und Datenquellenhinweise zu finden sind.                                                                                                                                                                                                                                                                                                                                                                                                             |
| <b>Plugin oder Erweiterung / Plugin or extension</b> | Ein Plugin kann alle Bestandteile der Karte erweitern. Eine Erweiterung (Plugin) kann u.a. den Inhaltsbereich, Kategorien, Werkzeuge erweitern.                                                                                                                                                                                                                                                                                                                                                                                               |
